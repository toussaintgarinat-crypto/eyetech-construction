# Tradelayer Intelligence iOS App

This directory contains the source code for the Tradelayer Intelligence iOS application.

## Project Structure

- `TradelayerIntelligenceApp/`: Main application source files (SwiftUI, ARKit, RealityKit)
- `Tests/`: Unit and UI tests
- `Resources/`: Assets, images, and other resources
- `Documentation/`: Project documentation

## Technologies Used

- Swift
- SwiftUI
- ARKit
- RealityKit

## Getting Started

1. Open the `.xcodeproj` or `.xcworkspace` file in Xcode.
2. Build and run the application on a simulator or a physical device.

## Features

- Professional Layer Management in AR
- Voice Commands Integration
- Spatial Analysis Visualization
